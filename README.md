Olive Oil QC Pro is a professional laboratory tool designed to analyze olive oil quality using UV absorbance indices (K232, K270/K268, Î”K) according to IOC standards.

The application:

Calculates UV quality indices automatically

Classifies olive oil (EVOO / VOO / Non-Virgin)

Integrates AI to generate structured French laboratory reports

Produces clean professional PDF reports with tables and conclusions

Stores full analysis history in SQLite

Optionally reads data directly from a spectrophotometer (USB serial)

Supports pesticide CSV import and AI interpretation

Language: French-only reports (professional laboratory style)
Target users: Olive oil producers, quality control labs, export companies.

Built with Streamlit + SQLite + OpenAI / DeepSeek + ReportLab.
